## Overview
This project analyzes IPL (Indian Premier League) data to extract insights about players, teams, and match outcomes using Python.  
It uses two datasets: `matches.csv` and `deliveries.csv`.

## Tools & Libraries
- Python (Google Colab)
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Top-performing batsmen and bowlers
- Team win percentages by season
- Toss decisions and match outcomes
- Venue-based performance trends
- Highest individual scores and economy rates

## Files
- `Project_IPL.ipynb` ‚Üí Python Notebook (main code)
- `matches.csv` ‚Üí Match-level dataset
- `deliveries.csv` ‚Üí Ball-by-ball dataset

## How to Run
1. Open the notebook in Google Colab or Jupyter.
2. Upload both CSV files to the same directory.
3. Run all cells to generate analysis and visualizations.

## Sample Visualizations
- Top 10 Run Scorers
- Most Successful Teams
- Matches Won by Toss Decision
- Season-wise Run Distribution
